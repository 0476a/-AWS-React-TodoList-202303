## 부족했던 점
- 자주 사용하는 내용들을 컴포넌트로 빼두지 못했다.
- 코드 정리가 잘 안되서 수정에 불편함이 있다.
- 제작 시간이 짧아서 부족한 부분이 많았다.
- 기능이 좀 부족했다.

## 보완해야할 점
- 자주 사용하는 것들을 컴포넌트로 따로 뺴서 관리
- 태그 위치 별로 이벤트를 정리하기
- 디자인 코드도 따로 폴더에 정리하기
- 완료 체크와 시간 지정 기능 추가
- 사진 변경과 삭제 기능 추가
- 다른 프로젝트도 진행 해보기
